## Capabilities

AG-UI enables:

- Agentic chat interfaces (headless and component-based)
- Generative UI functionality
- Synchronized application/agent state
- Human-in-the-loop interactions
- Oversight mechanisms (checkpoints, time travel)
- Thread management
